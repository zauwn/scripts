# Simple Wezterm Lua Config

## Featues
- Big scroll buffer
- Integrated title bar buttons
- Hack Nerd Font
- Tokyo Nigth/Day theme
- Support for setting tab title using keyboard shortcut (Shift+Ctrl+S)
- Alert on inactive tab when there is new/unseen output
